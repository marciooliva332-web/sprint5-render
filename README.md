# Dashboard de Análise de Inventário de Veículos

Este projeto faz parte do **Sprint 5** do curso de Análise de Dados da TripleTen. Ele consiste em uma aplicação web interativa desenvolvida em Python, utilizando a biblioteca **Streamlit**, para visualizar e analisar um conjunto de dados de anúncios de vendas de carros nos EUA.

## 🚀 Funcionalidades
O dashboard permite ao usuário realizar análises exploratórias rápidas através de:
* **Histograma interativo:** Visualização da distribuição de quilometragem (odômetro) dos veículos.
* **Gráfico de Dispersão:** Análise da relação entre o preço dos veículos e a sua quilometragem.
* **Interface Dinâmica:** Uso de checkboxes para alternar a exibição dos gráficos de forma independente.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Bibliotecas:** * Pandas (Manipulação de dados)
    * Streamlit (Criação do Web App)
    * Plotly Express (Gráficos interativos)
